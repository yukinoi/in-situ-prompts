# In-situ prompts guide attention in digital health design

This repository contains the code, case study materials, and the minimal dataset required to reproduce the results presented in this work.

## Repository contents

`notebooks/` — Jupyter notebook used to reproduce the reported results<br> 
`data/` — CSV files required to reproduce the reported baseline and sensitivity analyses<br>
`case_studies/` — structured case study documentation<br> 

## Reproducibility

The main analysis can be reproduced by running:

`notebooks/reproduce_results.ipynb`

The notebook can be executed directly in Google Colab:
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/yukinoi/in-situ-prompts/blob/main/notebooks/reproduce_results.ipynb)

The CSV files in `minimal dataset/` provide the minimal dataset required to reproduce the reported simulation outputs and summary statistics.
